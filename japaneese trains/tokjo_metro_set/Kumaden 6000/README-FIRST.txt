Petunjuk Penggunaan Rangkaian - Directions for Use

1. Struktur Folder - Folder Structure

Di dalam rangkaian ini tersedia folder tempat meletakkan semua nomor seri rangkaian Kumamoto 6000 (6101A, 6111A, 6211A hingga 6231A), di mana
untuk bertukar nomor seri cukup copy dan paste file num.bmp yang ada di dalam salah satu folder (6101A, 6111A, 6211A hingga 6231A) ke folder utama
(61X1A untuk 6101A-6111A dan 62X1A untuk 6211A-6231A).
Secara default, nomor seri yang dipasang adalah nomor 6108-6101 dan 6218-6211.

Untuk mengganti set 61XA ke 62XA dan sebaliknya, salin dan paste file Extensions.cfg dan Train.dat yang ada di masing-masing folder 61X1A atau 62XA ke folder rangkaian (Kumaden 6000).

This set includes all numbering arrangement for Kumamoto 6000 series cars (6101A. 6111A, 6211A to 6231A), which you can exchange those numbers
just by copy and paste the num.bmp files in an included folder (6101A. 6111A, 6211A to 6231A) to the main folder (61X1A for 6101A-6111A and 62X1A for 6211A-6231A). The rolling stock builder/author had
used the 6108-6101 and 6218-6211 number series by default.

To change 61XA train set to 62XA train set and vice-versa, copy and paste Extension.cfg and Train.dat file to the main folder (Kumaden 6000).

2. Penggunaan di BVE 2/4 - Using in BVE 2/4

Rangkaian ini secara default hanya mendukung penggunaan untuk openBVE. Jika anda memiliki BVE 2/4 dan ingin menggunakan
rangkaian ini di program tersebut, ubahlah header yang bertuliskan 'OPENBVE' menjadi 'BVE2000000' setelah membuka file Train.dat.

This set is only support openBVE by default. If you have BVE 2/4 versions and want to use this set in those programs,
please change the header 'OPENBVE' to 'BVE2000000' after opening the 'Train.dat' file.
